# jsxyhelu 的个人博客

这是基于 GitHub Pages 构建的个人博客网站。

## 技术栈

- HTML5
- CSS3
- GitHub Pages

## 本地预览

在本地浏览器中打开 `index.html` 文件即可预览网站效果。

## 部署方式

1. 推送代码到 GitHub 仓库 `jsxyhelu.github.io`
2. 在 GitHub 仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为发布源
4. 等待几分钟，网站即可通过 `https://jsxyhelu.github.io` 访问

## 如何更新博客

1. 编辑 `index.html` 文件，添加新的博客文章
2. 提交更改并推送到 GitHub
3. GitHub Pages 会自动更新网站

## 自定义域名

如果需要使用自定义域名，可以在 GitHub 仓库设置中配置：
1. 在 DNS 提供商处添加相应的 DNS 记录
2. 在 GitHub 仓库设置中添加自定义域名
3. 启用 HTTPS 支持
