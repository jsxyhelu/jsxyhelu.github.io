# 🌳 jsxyhelu 的绿色博客

这是基于 GitHub Pages 构建的个人博客网站，采用绿色和树木主题设计。

## 🌿 主题特色

- 🌲 **绿色主题**：采用自然绿色系配色，营造清新舒适的阅读环境
- 🌳 **树木元素**：融入树木和森林的视觉元素，体现自然与技术和谐共存的理念
- 🍃 **动态效果**：添加了树木摇摆、叶子飘落等动画效果，增强视觉体验
- 🌱 **响应式设计**：适配各种设备，提供良好的阅读体验

## 💻 技术栈

- HTML5
- CSS3 (包含渐变、动画、玻璃态效果)
- GitHub Pages
- 响应式设计

## 🎨 设计理念

本博客采用绿色和树木主题，寓意：
- **成长**：像树木一样在技术的土壤中茁壮成长
- **自然**：技术与自然和谐共存，追求可持续发展
- **生态**：开源精神如同森林生态系统，通过分享和协作让技术之树更加茂盛

## 🖥️ 本地预览

在本地浏览器中打开 `index.html` 文件即可预览网站效果。

## 🚀 部署方式

1. 推送代码到 GitHub 仓库 `jsxyhelu.github.io`
2. 在 GitHub 仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为发布源
4. 等待几分钟，网站即可通过 `https://jsxyhelu.github.io` 访问

## ✏️ 如何更新博客

1. 编辑 `index.html` 文件，添加新的博客文章
2. 保持绿色和树木主题的一致性
3. 提交更改并推送到 GitHub
4. GitHub Pages 会自动更新网站

## 🌐 自定义域名

如果需要使用自定义域名，可以在 GitHub 仓库设置中配置：
1. 在 DNS 提供商处添加相应的 DNS 记录
2. 在 GitHub 仓库设置中添加自定义域名
3. 启用 HTTPS 支持
