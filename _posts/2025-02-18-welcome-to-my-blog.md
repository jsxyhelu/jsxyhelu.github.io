## 为什么选择 GitHub Pages？

GitHub Pages 是一个免费的静态网站托管服务，非常适合用来发布个人博客。它的优势包括：

- **完全免费**：无需购买服务器或域名
- **自动部署**：推送代码后自动构建和发布
- **支持 Jekyll**：GitHub 原生支持 Jekyll 静态网站生成器
- **版本控制**：所有内容都在 Git 版本控制下
- **HTTPS 支持**：自动配置 SSL 证书

## 如何写博客？

使用 Jekyll 写博客非常简单，只需要：

1. 在 `_posts` 目录下创建 Markdown 文件
2. 文件名格式：`YYYY-MM-DD-标题.md`
3. 在文件顶部添加 Front Matter（元数据）
4. 使用 Markdown 语法编写内容
